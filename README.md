# 🎲 Dice Duel

**Dice Duel** is a simple two-player dice game created for **Boss Level Challenge 01 – The Dice Game**.

Each player receives a randomly generated dice roll. The player with the higher number wins the round, while matching numbers result in a draw.

## ✨ Features

* Two randomly generated dice rolls
* Automatic winner or draw result
* Roll Dice button to start a new round
* Responsive design for mobile and desktop screens
* Simple and beginner-friendly JavaScript logic
* No libraries or installation required

## 🚀 How to Run

1. Download or clone this repository.

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Open the `index.html` file in a web browser.

4. Click the **Roll Dice** button to play.

## 🎮 How the Game Works

1. Click the **Roll Dice** button.
2. Two random dice values are generated.
3. The player with the higher dice value wins.
4. If both values are equal, the round ends in a draw.
5. Click the button again to play another round.
